# niclsim.github.io

Personal portfolio site for Nicole Simmons, M.Ed.

**Live at:** https://niclsim.github.io

## What's here

- `index.html` — the portfolio site
- `Nicole_Simmons_Portfolio.pdf` — downloadable PDF version
- `images/` — embedded artifacts (MTSS framework, PD samples, STEM curriculum work)

## Updating the site

1. Edit `index.html` (or have Claude do it).
2. Open GitHub Desktop.
3. Review the changes in the diff view.
4. Commit with a short message ("update hero copy", "swap MTSS image", etc.).
5. Push to origin.
6. Site updates within ~30 seconds at https://niclsim.github.io.

## Tech notes

Pure static HTML/CSS — no build step, no dependencies. Self-contained. Hosted on GitHub Pages.
