Design Philosophy
Four moves that show up in every doc — because they're how adults actually learn software at work.
01
Warm, not directive
Peer-to-peer tone earns the read. "Fellow Teachers" — not "All Staff." Adult learners shut down at the first whiff of compliance training.
02
Multi-modal by default
Text + linked video + screenshots in every doc. Different learners, different preferences — same content, three modalities.
03
Reuse mechanisms baked in
Every guide points to compounding leverage — Share to Commons, parent course pattern, copy-from-prior-year. Each doc paid back the time it took to read.
04
Just-in-time, not training-calendar
Docs shipped before the friction hit. Teachers searched and found at the moment of need — not in a quarterly PD slot two months later.
Corporate Translation
Software rollout enablement · technical writing for adult learners · just-in-time microlearning · proactive support content · multi-modal documentation strategy. The same skill set that ships when an EdTech company rolls out a new product feature, an enterprise SaaS team onboards a new tool, or an L&D org shifts from calendar-based training to embedded enablement.

Case Study 02 · Quality System & Capability Build

Teacher Clarity Walkthrough Implementation

Admin-team monitoring system · Fisher & Frey clarity framework · Hattie ES 0.75

An admin-team walkthrough protocol that drove dramatic year-over-year academic growth, increased teacher capacity, and built continuous quality assurance into the system.

Challenge
Wide variability in teacher clarity drove outcome gaps. Walkthroughs typically read as surveillance — adoption stalled, data never became capability.
Action
  • Operated as an admin team — every classroom got structured observation.
  • Anchored to Fisher & Frey clarity framework (Hattie ES 0.75).
  • Randomized student selection + structured response capture.
  • Weekly feedback loops integrated into PLC + PD rhythms.
Result · Y1 → Y2
  • ELA +5.2% · Math +11% · Science +15% (schoolwide).
  • Math 8th grade nearly doubled (18% → 36%).
  • Increased teacher capacity through credible peer language.
  • System-level quality assurance, sustained.
Corporate Translation
Continuous quality assurance · capability monitoring at the IC level · evidence-based performance improvement · feedback systems that build skill, not enforce compliance.
